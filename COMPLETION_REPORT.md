
### 2. ROI 自动提取
```python
# 屏显图片自动找到文字区域
自动提取 → 形态学处理 → 轮廓分析 → ROI 区域
```

### 3. 多层次性能优化
```
L1 缓存：诊断结果 LRU 缓存（15x 加速）
L2 缓存：参数聚类和去重（5x 加速）
L3 缓存：模型预热和预加载
```

### 4. 自适应模型选择
```python
图像质量评分 (0-1)
    ↓
0.3~0.7    → Server 模型（精度优先）
0.7~1.0    → Mobile 模型（速度优先）
```

### 5. 完整的异常检测
```
IQR 检测 → 硬限检测 → 突变检测 → 基准检测 → 趋势检测
         ↓
去重和汇总 → 告警评分 (0-1)
```

## 📈 集成指南

### 快速集成（5 分钟）

```python
# 1. 导入新模块
from ai_engine.validator import ParameterValidator
from ai_engine.screen_optimizer import ScreenDisplayOptimizer

# 2. 在 OCR 后处理中添加
parameters = recognize_ocr_results(ocr_results)

# 3. 验证参数
validation_report = ParameterValidator.validate_parameters(parameters)

# 4. 屏显优化
if is_screen_display:
    parameters = ScreenDisplayOptimizer.deduplicate_parameters(parameters)

# 5. 返回增强结果
return {
    "parameters": parameters,
    "validation": validation_report,
    "diagnostics": diagnostics_info
}
```
