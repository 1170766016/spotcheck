# 工厂人员异常行为智能监控系统 — 技术评估报告

| 检测项 | 说明 |
|--------|------|
| 长时间站立不动 | 人员滞留/静止超过阈值 |
| 长时间低头 | 可能在玩手机或打瞌睡 |

| 多人聚集 | 区域人数超阈值 |

### 2. 关键缺口："工号识别"绑定到人

市面上现有方案的**人员身份识别**主要依赖：
- **人脸识别**（主流，但希望避免）
- **工服颜色/款式区分**（只能区分工种，不能区分个人）
- **RFID标签/工牌**（需要额外硬件）

**"通过识别衣服上的号码来锁定具体个人"这个方案，目前没有现成的商业产品直接支持。** 这是一个需要定制开发的环节。

## 三、"号码衣服"方案的技术可行性分析

### 核心技术链路

```
摄像头画面 → 人体检测 → 号码区域定位 → OCR号码识别 → 绑定人员身份 → 行为分析 → 异常告警
```

### 关键技术难点

| 难点 | 说明 | 难度 |
|------|------|------|
| **号码可见性** | 工人转身、弯腰、被遮挡时号码不可见；工厂环境光线、褶皱、污损影响识别 | **高** |
| **号码识别精度** | 普通200万像素摄像头在5米外很难看清衣服上的数字，需要**400万以上像素**或近距离部署 | **中高** |
| **持续追踪** | 号码只在特定角度可见，需要结合人体ReID（外观特征重识别）做跨帧追踪 | **高** |
| **多人重叠** | 聚集场景下人员互相遮挡，号码难以读取 | **中** |

### 老板方案的真实成本评估

想法是"穿号码衣服 → 省去高清摄像头和人脸采集"，但实际上：

- **号码识别对摄像头分辨率要求反而更高**：看清衣服上的数字比检测人体轮廓需要更高的像素密度
- **号码识别不如人脸识别稳定**：人脸有成熟的深度学习方案，而衣服号码受姿态、遮挡、光照、褶皱影响大
- **结论**：号码衣服方案**并不能显著降低硬件成本**，反而增加了算法定制开发成本

## 四、推荐技术方案

### 方案A：商业成品方案（推荐优先考虑）

直接采购燧机科技或类似厂商的AI行为分析系统：

| 项目 | 预估成本 |
|------|----------|
| AI智能摄像头（含算法） | ¥6,000~20,000/路 |
| 后台管理平台 | ¥10,000~50,000（一次性） |
| 部署实施 | ¥5,000~10,000/路 |
| 假设10个监控点位 | **总计约 ¥15~30万** |

**优点**：开箱即用，行为检测成熟（离岗、摔倒、聚集、睡岗等）

**缺点**：身份识别仍需人脸或RFID，号码识别需定制

---

### 方案B：定制开发方案

自行或委托开发整套系统：

| 技术模块 | 技术选型 | 说明 |
|----------|----------|------|
| 人体检测 | YOLOv8/v11 | 成熟开源，免费 |
| 姿态估计 | OpenPose / MediaPipe / YOLO-Pose | 检测摔倒、低头、站立姿态 |
| 行为分析 | ST-GCN / 规则引擎 | 时序动作识别 |
| 人员追踪 | ByteTrack / DeepSORT | 多目标跟踪 |
| 号码识别 | OCR（PaddleOCR）+ 号码区域检测 | **需定制训练** |
| 人员ReID | 外观特征重识别 | 跨摄像头/跨帧身份关联 |
| 边缘计算硬件 | 瑞芯微RK3588/NVIDIA Jetson | ¥2,000~5,000/台 |
| 摄像头 | 400万像素网络摄像头 | ¥300~800/台 |

| 项目 | 预估成本 |
|------|----------|
| 算法开发（含号码识别） | ¥15~30万（外包）/ 2-4个月（自研） |
| 后台系统开发 | ¥5~10万 |
| 硬件采购（10路） | ¥3~8万 |
| 数据采集与标注 | ¥2~5万 |
| **总计** | **约 ¥25~53万** |
| **开发周期** | **3~6个月** |

---

### 方案C：混合方案（推荐最优解）

| 组成 | 方案 |
|------|------|
| 行为检测 | 采购商业AI摄像头（燧机科技等），开箱即用 |
| 身份识别 | **RFID工牌/UWB定位标签**（¥50~200/个）替代号码衣服 |
| 系统整合 | 将RFID定位数据与摄像头行为数据融合 |

**RFID方案优势**：每个工牌绑定一个人，无需看清脸或号码，成本远低于高清摄像头，且技术成熟。

| 项目 | 预估成本 |
|------|----------|
| AI行为摄像头 × 10 | ¥6~20万 |
| RFID读写器 + 天线 | ¥2~5万 |
| RFID工牌 × 100人 | ¥0.5~2万 |
| 系统整合开发 | ¥5~10万 |
| **总计** | **约 ¥14~37万** |

## 七、法律与合规风险提示

1. **个人信息保护法（PIPL）**：即使不用人脸识别，持续监控员工行为并记录可能涉及个人信息处理
2. **劳动法**：需要告知员工并获得同意，监控数据的使用范围需要明确
3. **员工关系**：过度监控可能引发员工抵触，建议以"安全管理"而非"防偷懒"为定位

## 八、总结建议

| 维度 | 建议 |
|------|------|
| **最快落地** | 采购燧机科技等商业方案，1~2周可部署，约¥15~30万/10路 |
| **身份识别** | 建议用 **RFID工牌** 替代号码衣服方案，技术成熟、成本低、可靠性高 |
| **号码衣服方案** | 可行但不推荐，号码识别精度受环境影响大，且需要定制开发，反而增加成本和周期 |
| **自研方案** | 如果公司有AI团队，3~6个月可完成，但综合成本高于商业方案 |
| **推荐方案** | **方案C（商业AI摄像头 + RFID工牌 + 整合开发）**，性价比最高 |
