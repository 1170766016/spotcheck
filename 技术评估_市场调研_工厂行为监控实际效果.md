## 二、他们检测什么？和我们的需求一样吗？

### 2.1 市面上主流的检测功能

| 检测功能 | 市面覆盖率 | 我们的需求 | 匹配度 |
|----------|-----------|-----------|--------|
| 安全帽/工服穿戴检测 | 几乎所有厂商都有 | 不直接需要 | - |
| 离岗/空岗检测 | 几乎所有厂商都有 | 类似（长时间不在工位） | ✅ 高 |
| 睡岗检测 | 大部分厂商有 | 类似（长时间低头/不动） | ✅ 高 |
| 抽烟检测 | 大部分厂商有 | 不需要 | - |
| 打电话/玩手机检测 | 大部分厂商有 | 类似（长时间低头） | ⚠️ 部分匹配 |
| 摔倒检测 | 部分厂商有 | **核心需求** | ⚠️ 有但不够多 |
| 区域闯入检测 | 几乎所有厂商都有 | 不需要 | - |
| 人员聚集/超员检测 | 部分厂商有 | **核心需求** | ⚠️ 有但不够多 |
| 长时间站立不动 | **很少厂商直接提供** | **核心需求** | ❌ 几乎没有 |
| 打架/暴力行为检测 | 部分厂商有 | 不需要 | - |
| 操作动作顺序合规 | 部分厂商有（燧机/海研） | 不需要 | - |

### 2.2 关键发现

> **市面上的系统主要围绕"安全生产"设计（安全帽、工服、抽烟、闯入），不是围绕"防偷懒"设计的。**

我们的核心需求中：
- **多人聚集**：有厂商提供（燧机科技有"超员检测"）
- **摔倒检测**：有厂商提供（燧机科技、算丰科技）
- **长时间站立不动**：**几乎没有厂商直接提供这个功能**
- **长时间低头**：没有直接提供，但"睡岗检测"和"玩手机检测"有部分重叠
- **号码衣服识别身份**：**没有任何厂商提供**

### 3.2 效果总结

| 检测类型 | 实际精度 | 误报率 | 实用性评价 |
|----------|---------|--------|-----------|
| **安全帽/工服穿戴** | 95~99% | 1~3% | ⭐⭐⭐⭐⭐ **非常成熟，直接可用** |
| **区域闯入/人数统计** | 93~97% | 2~5% | ⭐⭐⭐⭐ **成熟可用** |
| **离岗/空岗检测** | 实际效果好 | 低 | ⭐⭐⭐⭐ **实用，煤矿实测月均7次→1次** |
| **违规操作检测** | 94~95% | 2~3% | ⭐⭐⭐⭐ **可用，但需要针对性训练** |
| **睡岗检测** | 69~90% | 8~12次/千小时 | ⭐⭐⭐ **可用但不够精准，误报偏多** |
| **摔倒检测** | 90%+ | 中等 | ⭐⭐⭐ **可用，蹲下vs摔倒的区分仍是挑战** |
| **低头/玩手机** | 70~85% | 较高 | ⭐⭐ **可用但误报多（低头干活 vs 玩手机难区分）** |
| **长时间站立不动** | 无公开数据 | 未知 | ⭐ **市面几乎没有成熟产品** |

### 3.3 实际使用中的主要问题

根据多个部署案例的反馈，**实际使用中最常见的问题**：

1. **误报率比实验室高 2~5 倍**：光照变化、遮挡、摄像头角度不佳是主因
2. **睡岗检测误报严重**：揉眼、低头记笔记、查阅文件经常被误判为睡岗
3. **需要反复调参**：每个工厂环境不同，部署后需要 1~3 个月的调优期
4. **过度敏感会降低信任度**：误报太多管理人员会忽略告警，失去意义
5. **夜间/弱光环境效果下降明显**：需要额外补光或红外摄像头

## 四、"防偷懒"这个场景有人做吗？

### 4.1 现状

**直接以"防偷懒"为卖点的视觉监控系统，市面上几乎没有。**

原因：

| 因素 | 说明 |
|------|------|
| **法律风险** | 《个人信息保护法》对工作场所监控有明确限制，"防偷懒"定位容易引发劳动纠纷 |
| **员工抵触** | 多家企业因过度监控引发员工不满和舆论危机 |
| **效果存疑** | "站在工位不动" ≠ "在偷懒"，很多精密制造工位本身就需要长时间静止观察 |
| **商业定位** | 厂商更愿意包装为"安全生产管理"，避免负面标签 |

### 4.2 相近的替代产品

虽然"摄像头防偷懒"很少，但**电脑端的行为监控**在办公场景已经很成熟：

| 产品 | 功能 | 场景 |
|------|------|------|
| WorkWin | 监控员工电脑使用（软件使用时长、网页浏览、效率分析） | 办公室 |
| 各类上网行为管理 | 监控网络流量、应用使用、文件操作 | 办公室 |
| 钉钉/企微打卡+工位传感器 | 在离岗统计 | 办公室 |

**这些都不适用于工厂车间场景。**

### 4.3 工厂实际怎么管"偷懒"？

根据调研，工厂实际管理"偷懒"的方式：

| 方式 | 普遍程度 | 说明 |
|------|---------|------|
| **产量/计件考核** | ⭐⭐⭐⭐⭐ | 最主流 — 看产出结果而非过程 |
| **班组长巡查** | ⭐⭐⭐⭐ | 传统方式，人工管理 |
| **普通监控回放** | ⭐⭐⭐⭐ | 大部分工厂有摄像头，但靠人工查看 |
| **产线节拍管控** | ⭐⭐⭐ | 流水线有标准节拍，慢了就知道 |
| **AI行为监控** | ⭐ | 极少数工厂试点，且以安全管理为主 |

## 五、核心问题的直白回答

### Q1：有工厂在用这种系统吗？
**有**，但主要是"安全生产管理"（安全帽、工服、抽烟、闯入），不是"防偷懒"。

### Q2：效果怎么样？精准吗？
- **简单的检测项**（安全帽、人数统计、区域闯入）：**精准，95%+，可直接用**
- **中等难度的检测项**（离岗、睡岗）：**可用但误报偏多，69~90%**，需要反复调参
- **高难度的检测项**（低头玩手机、摔倒）：**勉强可用，70~85%**，蹲下 vs 摔倒、低头干活 vs 玩手机很难区分
- **"长时间站立不动"**：**没有成熟产品**，需要定制开发

### Q3：是否实用？
- **安全管理场景**：**实用**，已有大量落地案例，确实能减少安全事故
- **防偷懒场景**：**不太实用**，原因是：
  - "站着不动"不等于偷懒（精密制造很多工位就是需要静止操作）
  - 误报太多会导致管理人员疲劳，反而忽略真正的问题
  - 容易引发员工抵触和法律风险

### Q4：有没有提供商直接提供这种功能？
- **燧机科技**：产品线最全（离岗/睡岗/摔倒/聚集/超员都有），但**没有"长时间站立不动"检测**
- **海康威视/大华**：有智慧工厂方案，但是项目制，起步价高
- **算丰科技/极视角**：有算法授权，可以按路数购买
- **没有任何厂商**提供"号码衣服识别身份"的功能

### Q5：有没有工厂已经引入了"防偷懒"系统？效果如何？
**据公开资料，没有工厂以"防偷懒"为明确目标部署AI视觉监控系统。** 部署了的工厂都是以"安全生产管理"为目标，效果主要体现在：
- 单产线年减少损失约 25 万元（较人工巡检）
- 空岗事件从月均 7 次降到 1 次
- 系统处理效率相比人工提升 300%
- 一年可节约人工成本 162 万+（大型厂区）

## 六、给我们的建议

### 6.1 如果要做，怎么定位

**建议包装为"安全生产智能管理系统"而非"防偷懒系统"**：

| 对外说法 | 对内实际效果 |
|----------|------------|
| "保障员工安全，及时发现摔倒、聚集等异常" | 顺便检测了人员静止/聚集 |
| "规范作业流程，减少安全事故" | 顺便监控了人员行为 |
| "提升生产管理数字化水平" | 顺便获得了人员行为数据 |

### 6.2 采购建议

如果决定先买一套试试效果：

| 推荐 | 厂商 | 理由 |
|------|------|------|
| **首选** | 燧机科技 | 产品线最全，价格透明（¥1,480~19,800/路），有现成产品可买 |
| **备选** | 算丰科技 | 边缘AI盒子，可接现有摄像头，¥6,000/路起 |
| **大型项目** | 海康威视/大华 | 如果预算充足、规模大，可以找他们做整体方案 |

### 6.3 先买后研的策略

1. **先花 ¥6,000~20,000 买一路燧机科技的AI摄像头试试**
2. 在自己工厂实际跑 1~2 个月，看看：
   - 实际误报率是多少
   - 管理人员是否觉得有用
   - 员工反应如何
   - 哪些检测项真的有效，哪些是噱头
3. **根据试用结果决定**：是继续采购还是自研

这样做的好处：**花最少的钱，最快验证可行性，避免盲目投入。**
